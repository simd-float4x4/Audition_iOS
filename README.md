
# オーディションアプリ

## プロダクト概要

* 芸能事務所向けのオーディションアプリ。ユーザーのフロントアプリと、クライアントアプリでダッシュボードを具備する。
* このリポジトリでは、フロント（iOS）を取り扱う。

## ソフトウェア要件

| 項目 | 設定内容 |
| :--- | :--- |
| **Xcode Version** | `Xcode 26.0 (17A324)` |
| **Compiler Version** | `Swift 6` |
| **Language Mode** | `Swift 6` |
| **Minimum Deployments** | `iOS 18.2 〜` |
| **Supported Destinations** | `iPhone`, `iPad` |


## 実装方針
* アーキテクチャ, コードの綺麗さよりも雑でスピード感のある実装をする。
* チーム開発ではないため、途中アンチパターンがあろうが表記揺れがあろうが、最終的に完成した際の見た目が綺麗で保守性があればそれでよいこととする。
* 生成AIを使用してもよいが、何を解決したくて、どう考えて、どう解消したのか、思考のプロセスが残るようにする。
    * 可能であれば使用したモデルとプロンプトを残しておくこと。
